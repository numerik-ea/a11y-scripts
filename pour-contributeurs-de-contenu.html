Pour ajouter les bookmarklets dans votre barre des favoris, suivez les étapes ci-dessous :

  1. Cliquez sur les liens des bookmarklets que vous souhaitez ajouter.
  2. Maintenez le clic gauche de la souris enfoncé sur le lien.
  3. Faites glisser le lien vers la barre des favoris de votre navigateur.
  4. Relâchez le bouton de la souris pour y déposer le bookmarklet.
  5. Voilà ! Vous pouvez maintenant utiliser les bookmarklets pour vérifier que votre contenu est accessible.
